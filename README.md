# 學生成績管理系統

以 C++ 實作的學生成績管理系統，展示 STL 容器與 template 函式的應用。

## 編譯與執行

```bash
g++ -std=c++17 main.cpp -o main
./main
```

## 功能列表

| 編號 | 功能 | 說明 |
|------|------|------|
| 1 | 新增學生 | 輸入學號、姓名、成績；學號重複時詢問是否覆蓋 |
| 2 | 列出所有學生 | 以表格形式顯示所有學生資料 |
| 3 | 依成績排序 | 使用 `std::sort` 依成績由高到低排序 |
| 4 | 查詢學生 | 輸入學號查詢學生資料 |
| 5 | 統計成績 | 計算平均、最高分、最低分、及格與不及格人數 |
| 0 | 離開 | 結束程式 |

## Template 函式

統計功能使用了兩個自訂的 template 函式：

- `getMax<T>(T a, T b)` — 回傳較大的值
- `getMin<T>(T a, T b)` — 回傳較小的值

## 資料結構

使用 `vector<Student>` 作為主要資料結構，`Student` 結構包含：

- `id` (string) — 學號
- `name` (string) — 姓名
- `score` (int) — 成績
